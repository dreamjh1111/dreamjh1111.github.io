# OpenClaw Introduction Article Design

## Goal

`#05 AI/#01 Open-Claw` 시리즈의 `00`번 글로, OpenClaw를 처음 보는 독자에게

- OpenClaw가 무엇인지
- 왜 갑자기 많이 언급되는지
- 기존 챗봇과 무엇이 다른지
- 대충 어떤 구조로 돌아가는지
- 누구에게 맞고, 왜 바로 업무에 넣기엔 조심해야 하는지

를 짧고 이해하기 쉽게 설명한다.

## Audience

- OpenClaw를 처음 보는 입문자
- 설치 글부터 보기 전에 개념을 먼저 잡고 싶은 독자
- "그냥 ChatGPT랑 뭐가 다른데?"가 궁금한 독자

## Structure

1. 도입: OpenClaw를 한 문장으로 설명
2. 왜 주목받는가: `The AI that actually does things`
3. 무엇이 다른가: 일반 챗봇/에이전트 프레임워크와의 차이
4. 어떻게 돌아가나: Gateway, Control UI, channels, memory, heartbeat
5. 누가 쓰면 좋은가 / 아직 이른 사람은 누구인가
6. 다음 글 연결: Ubuntu 설치편

## Constraints

- 설치 글보다 짧게 유지
- 공식 사이트와 나무위키 링크를 참고 링크에 포함
- 한국어 커뮤니티 반응도 반영
- 이미지 1개 + Mermaid 1개 포함
- 과장하지 않고 보안/운영 난이도도 같이 언급
